# Homework

This directory contains homework exercises as Jupyter notebooks, along with their solutions.

## Structure

Each assignment lives in its own sub-directory named `hwNN_<short_topic>` where `NN` is the
zero-padded assignment number (e.g. `hw01_leaky_integrate_and_fire`).

```
homework/
└── hwNN_<topic>/
    ├── exercise.ipynb   ← notebook for students (solutions removed)
    └── solution.ipynb   ← notebook with complete solutions
```

Use [`hw_template/`](hw_template/) as a starting point for new assignments.

## Running the Notebooks

You can run each notebook:

- **Locally** – install [JupyterLab](https://jupyter.org/) (`pip install jupyterlab`) and open the `.ipynb` file.
- **In the browser** – click the *Open in Colab* badge at the top of any notebook to run it instantly in [Google Colab](https://colab.research.google.com/) with no local setup required.

> **Instructors:** when copying `hw_template/` to a new assignment folder, update the Colab badge URL in the first markdown cell of each notebook so it points to the correct file path on the `main` branch.
