# Lectures

This directory contains slides and recordings for each lecture.

## Structure

Each lecture lives in its own sub-directory named `NN_<short_topic>` where `NN` is the
zero-padded lecture number (e.g. `01_introduction`).

```
lectures/
└── NN_<topic>/
    ├── README.md    ← recording link and summary of topics covered
    └── slides.*     ← slide deck (PDF, PPTX, …)
```
